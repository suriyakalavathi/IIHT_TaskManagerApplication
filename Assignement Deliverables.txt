Few Steps on how to run the solution
Keep MYSQL up and running
Keep TaskManagerApplication backend service up and running 
execute ng serve from UI project path in terminal to run the UI app. 
Once the app is running in local host,hit http://localhost:4200/task/add to access task manager app to task.
